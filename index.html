Welcome to CharityHub.

This project was made to keep in mind all the troubles organisations go through to reach out to people. Therefore, CharityHub makes it really easy for organisations to reach out to all the generous donors.

How This is done?

The project contains a web app as well as this mobile app.The web app will be uploaded on GitHub very soon. While this mobile app solely for donors or the common people, the web app is for the organisations to register themselves as well as keep posting about their needs. The mobile app (for now Android, soon for iOS) is used to keep in touch with the organisations and see their needs. You can donate with the help of the app itself. The app will be giving you any latest notifications from any of the registered organisations.
